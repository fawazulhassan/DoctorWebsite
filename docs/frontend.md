## 🎨 DESIGN SYSTEM

### **Color Palette**

| Role | Color | Usage |
|---|---|---|
| Primary | Teal / Medical Blue-Green | Main brand, CTAs, headers |
| Secondary | Soft Green | Pediatric section accent |
| Accent | Warm Orange | Highlight, CTA hover |
| Emergency | Red (#E53E3E) | Emergency button, alerts |
| Success | Green (#38A169) | Confirmed status |
| Warning | Amber (#ECC94B) | Pending status |
| Neutral | Gray shades | Text, borders, backgrounds |
| Background | White / Off-white | Page backgrounds |

### **Typography**

- **English:** Inter or Poppins (clean, medical-appropriate)
- **Urdu:** Noto Nastaliq Urdu (for bilingual labels)
- **Headings:** Bold weight, clear hierarchy (H1–H4)
- **Body:** Regular weight, 16px base, high contrast

### **Spacing & Layout**

- Grid: 12-column grid
- Max container width: 1280px
- Gutters: 24px (desktop), 16px (mobile)
- Section padding: 80px (desktop), 40px (mobile)

### **Component States**

All interactive components must have:
- Default
- Hover
- Active / Pressed
- Focused (accessible outline)
- Disabled
- Loading
