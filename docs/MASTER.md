**Distance Calculation:**
```javascript
const calculateDistance = async (destLat, destLng) => {
  const service = new google.maps.DistanceMatrixService();
  const origin = { lat: 31.5822, lng: 74.3596 }; // Harbanspura coords
  const destination = { lat: destLat, lng: destLng };
  
  service.getDistanceMatrix({
    origins: [origin],
    destinations: [destination],
    travelMode: 'DRIVING',
  }, (response, status) => {
    if (status === 'OK') {
      const distanceKm = response.rows[0].elements[0].distance.value / 1000;
      const zone = getZone(distanceKm);
      const price = calculatePrice(zone);
      setDistance(distanceKm);
      setZone(zone);
      setPrice(price);
    }
  });
};

const getZone = (km) => {
  if (km <= 5) return 'green';
  if (km <= 10) return 'yellow';
  if (km <= 15) return 'red';
  return 'out-of-range';
};

const calculatePrice = (zone) => {
  const baseFee = 2000; // consultation
  const transportFees = {
    green: 300,
    yellow: 500,
    red: 700,
  };
  return baseFee + (transportFees[zone] || 0);
};
```

```javascript
// JazzCash payment initiation
const initiateJazzCashPayment = async (appointmentData) => {
  const payload = {
    pp_Version: '1.1',
    pp_TxnType: 'MWALLET',
    pp_Language: 'EN',
    pp_MerchantID: process.env.JAZZCASH_MERCHANT_ID,
    pp_Password: process.env.JAZZCASH_PASSWORD,
    pp_TxnRefNo: `TXN${Date.now()}`,
    pp_Amount: appointmentData.totalAmount * 100, // Convert to paisa
    pp_TxnCurrency: 'PKR',
    pp_TxnDateTime: getCurrentDateTime(),
    pp_BillReference: appointmentData.appointmentNumber,
    pp_Description: `Appointment with Dr. ${appointmentData.doctorName}`,
    pp_TxnExpiryDateTime: getExpiryDateTime(),
    pp_ReturnURL: `${process.env.CLIENT_URL}/payment/return`,
    pp_SecureHash: '', // Generated below
    pp_MobileNumber: appointmentData.patientPhone,
    pp_CNIC: '', // Optional
    ppmpf_1: appointmentData.appointmentId, // Custom field
  };
  
  // Generate secure hash
  payload.pp_SecureHash = generateSecureHash(payload);
  
  return payload;
};

// Secure hash generation (HMAC SHA256)
const generateSecureHash = (data) => {
  const sortedString = Object.keys(data)
    .filter(key => key !== 'pp_SecureHash')
    .sort()
    .map(key => data[key])
    .join('&');
  
  const hash = crypto
    .createHmac('sha256', process.env.JAZZCASH_INTEGRITY_SALT)
    .update(sortedString)
    .digest('hex');
  
  return hash.toUpperCase();
};

// Payment verification on return
const verifyJazzCashPayment = (callbackData) => {
  const receivedHash = callbackData.pp_SecureHash;
  const calculatedHash = generateSecureHash(callbackData);
  
  if (receivedHash === calculatedHash) {
    if (callbackData.pp_ResponseCode === '000') {
      // Payment successful
      return { success: true, transactionId: callbackData.pp_TxnRefNo };
    } else {
      // Payment failed
      return { success: false, error: callbackData.pp_ResponseMessage };
    }
  } else {
    // Hash mismatch - possible tampering
    return { success: false, error: 'Security verification failed' };
  }
};
```

**Emergency Intake Form:**
- **Patient Quick Info:**
  - Name*
  - Phone*
  - Age
  - Location (Lahore/Kasur)*
- **Emergency Details:**
  - Brief description* (500 chars)
  - Severity (dropdown): Mild / Moderate / Severe
  - Patient condition (conscious, breathing difficulty, etc.)
  - For which doctor (Psychiatrist/Pediatrician)*
- **Preferred Contact Method:**
  - Call me immediately
  - WhatsApp video call
  - Schedule earliest available slot

**Backend Processing:**
- Form submission triggers:
  1. **Instant WhatsApp message to doctor:**
     ```
     🚨 EMERGENCY CONSULTATION REQUEST 🚨
     
     Patient: [Name]
     Phone: [Phone]
     Age: [Age]
     Location: [Lahore/Kasur]
     Severity: [Severe]
     
     Condition:
     [Patient's description]
     
     Requested: [Call / WhatsApp Video]
     
     Please contact patient ASAP!
     ```
  
  2. **SMS alert to doctor's phone**
  
  3. **Email notification**
  
  4. **Admin dashboard alert:**
     - Red notification badge
     - Popup alert
     - Sound notification (optional)
  
  5. **Auto-call to patient** (optional - using Twilio):
     - "Your emergency request has been received. Dr. [Name] will contact you within 5 minutes."

**Doctor Response:**
- Doctor receives all alerts
- Calls/video calls patient directly
- Or schedules urgent appointment (within next 2 hours)
- Marks request as "Attended" in system

- Higher fee (e.g., 1.5x or 2x regular fee)
- Mentioned in form clearly
- Payment can be later (after consultation)

- Admin can view all emergency requests
- Status: Pending / Attended / Follow-up needed
- Response time tracking
- Outcome recording

- Even outside clinic hours, form available

- Automated response: "Emergency received. On-call doctor will respond within 15 minutes."

**Follow-up:**
- If emergency consultation via call, book follow-up appointment
